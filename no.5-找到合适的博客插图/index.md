## 免费无版权的图片网站

### NO.1 Pixabay

{{< style "text-align:center" >}}

{{< link href="https://pixabay.com/" content=Pixabay官网 title="进入Pixabay" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/09/ocsm4qxPd3XiuOS.png" alt="图片1" caption="Pixabay官网" title="Pixabay官网" >}}

这个网站是我看了这么多网站中仅次于`Shutterstock`的，Pixabay的图片质量和数量都不错，虽然搜索的第一结果没有`Shutterstock`那么“耀眼”，毕竟对免费网站来说做到这样已经很良心了，无论是运营号还是自己写博客玩，都够用了。顺带提一句：`Shutterstock`是真的贵！图片是真的好！

- <strong>优点</strong>：免费、质量高、无需注册直接下载
- <strong>缺点</strong>：英文描述准确才能精准搜索、内容里面有`Shutterstock`的广告看着眼馋

<br/>

### NO.2 Unsplash 

{{< style "text-align:center" >}}

{{< link href="https://unsplash.com/" content=Unsplash官网 title="进入Unsplash" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/8rTRphluvnHVzbd.png" alt="图片2" caption="Unsplash官网" title="Unsplash官网" >}}

Unsplash无疑是所有无版权精美图片的最大聚集地，不用注册直接下载，很方便，用的人也多。但为什么排第二，是因为搜索图片内容很难围绕输入的关键词出现精确的图片，虽然质量高，但就写作内容指向性很强的文章而言搜索比较费时。不过Unsplash的图拿来作为文章插图是非常不错的。

- <strong>优点</strong>：不用注册直接下载、质量高、多尺寸选择、支持部分中文搜索
- <strong>缺点</strong>：搜索结果不精确、内容指向性不强

<br/>

### NO.3 Pexels

{{< style "text-align:center" >}}

{{< link href="https://www.pexels.com/zh-cn/" content=Pexels官网 title="进入Pexels" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/UuQsXzCfSY3RAlZ.png" alt="图片3" caption="Pexels官网" title="Pexels官网" >}}

Pexels的主页设计很干净简洁，配色也很好看，比千图网这类的网站看着舒服多。搜索指定关键词的内容也较精确，但相对Pixabay稍差，二者其实也能互补。另一大优点就是这个网站可以完全用中文关键词进行搜索。其次，图片下载也是不用注册，可以选择不同尺寸。

- <strong>优点</strong>：支持中文搜索、不用注册、多尺寸、内容相对精确
- <strong>缺点</strong>：内容指向性上还是不如Shutterstock、Pixabay

<br/>

### NO.4 New Old Stock

{{< style "text-align:center" >}}

{{< link href="https://nos.twnsnd.co/" content=New-Old-Stock官网 title="进入New Old Stock" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/u2UCewG7qrg1dEa.png" alt="图片4" caption="New Old Stock官网" title="New Old Stock官网" >}}

这个网站真的让人赏心悦目。每张图片都有它背后的故事，而每张老照片背后的故事就更让人神往。站内所有图片都是一些作者上传的高清老照片，感觉像是[经过数码处理的]^(不专业的猜测)，所以虽然老，但很清晰。内容很精致，适合一些叙事类文章做插图用，很有感觉。

- <strong>优点</strong>：内容精致、图片质量高
- <strong>缺点</strong>：只能有什么图用什么图，能不能找到合适的全看缘分

<br/>

### NO.5 Pinterest

{{< style "text-align:center" >}}

{{< link href="https://www.pinterest.com/" content=Pinterest官网 title="进入Pinterest" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/pwNY9OHxrcBv5RT.png" alt="图片5" caption="Pinterest官网" title="Pinterest官网" >}}

这个网站是从各类平台收集图片，用户把自己脸书或者推特上面的图贴在这免费让大家使用，内容质量还是挺高的，但是不支持特定尺寸的下载，点击图片就到对应的出处位置进行下载。网站会根据你收藏的喜好进行推荐，需要调教，有点麻烦，偶尔逛逛找找灵感还不错。

- <strong>优点</strong>：支持中文搜索、图片质量高
- <strong>缺点</strong>：需要注册、内容推荐有一定的过程、下载的图片可能有水印

<br/>

### NO.6 LibreStock

{{< style "text-align:center" >}}

{{< link href="https://librestock.com/" content=LibreStock官网 title="进入LibreStock" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/KpNGQs9LAqwyV21.png" alt="图片6" caption="LibreStock官网" title="LibreStock官网" >}}

- <strong>缺点</strong>：图片质量不高、不支持中文搜索、精确度不高

### NO.2 FLATICON

{{< style "text-align:center" >}}

{{< link href="https://www.flaticon.com/" content=FLATICON官网 title="进入FLATICON" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/v5EXjfgMDHb6OY7.png" alt="图片8" caption="FLATICON官网" title="FLATICON官网" >}}

这个网站内容也很齐全，只要搜索的英文准确，图标结果也很准确。图标内容元素更多，适合用来做文章的插图或者给图片增加一些层次。但是对网站的访问和下载有要求，需要科学上网，有一定门槛。不过现在科学上网已经不是事了。

### NO.3 stories

{{< style "text-align:center" >}}

{{< link href="https://stories.freepik.com/" content=stories官网 title="进入stories" >}}

{{< /style >}}

{{< image src="https://i.loli.net/2020/05/10/wjJtZ3dFvzRMGKW.png" alt="图片9" caption="stories官网" title="stories官网" >}}

这一个与下一个网站我如果早发现的话，估计做PPT的水平要上一个台阶。里面的内容主要以插画为主，适合用来表达一些关键词的意思，突出文章重点。所有的图片内容都可以更改，而且有很方便的交互设计，鼠标点几下就可以更改内容颜色、动画、不同图层等等。是做PPT的大杀器！

- <strong>优点</strong>：无需注册即可下载、可自定义、内容丰富
